# Lenith Block

**Automation Engineer · Full Stack Developer · AI Systems**

Construyo sistemas que generan resultados: captación de leads, automatización de seguimiento y páginas que convierten. Trabajo con negocios que quieren más clientes y menos tareas manuales.

---

## Qué hago

**Sistemas de generación de leads**
Scraping y enriquecimiento de datos, clasificación automática de prospectos, base de datos lista para accionar.

**Automatización de seguimiento**
Emails y mensajes automáticos, flujos en n8n, integración con CRM y APIs.

**Landing pages que convierten**
Diseño enfocado en conversión, carga rápida y responsive, copy y estructura estratégica.

**Integraciones a medida**
Conexión entre herramientas vía APIs, automatización de procesos internos, eliminación de tareas repetitivas.

---

## Proyectos

**Pipeline de leads automatizado**
Captura → limpieza → contacto automático. Menor trabajo manual, mayor velocidad de cierre.

**Automatizaciones con n8n**
Workflows end-to-end con integraciones a múltiples servicios.

**Web + JS + Automatización**
Landing con sistema de captación integrado y seguimiento automático post-lead.
